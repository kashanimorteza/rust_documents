## cargo
General
```bash
cargo --help
cargo --version
```

Create a package as a binary
```bash
    cargo new my_project_1 
```

Create a package as a library
```bash
    cargo new my_library_1 --lib
```

Add external package
```bash
    cargo add random
```


Other
```bash
cargo run
cargo run -q
cargo run --release
cargo build
cargo build --release
cargo fmt
```
